Cấu trúc project
admin/       : Trang quản trị
assets/      : CSS, JS, hình ảnh
config/      : Cấu hình database
inc/         : File include
vendor/      : Thư viện composer
Cài đặt project

Clone project:

git clone https://github.com/phuong08102004/PHP.git

Copy project vào:

xampp/htdocs

Import database:

honeybee_schema.sql

Chạy project:

http://localhost/honeybee
